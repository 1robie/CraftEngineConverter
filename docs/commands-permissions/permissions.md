This page provides a comprehensive reference for all permissions available in **CraftEngineConverter**. Each permission is detailed with its description and the commands it is associated with.

## Available Permissions

1. `craftengineconverter.command.use`
   - **Description**: Grants access to the main CraftEngineConverter command.
   - **Associated Command**: `/craftengineconverter` or `/cec`

2. `craftengineconverter.command.reload`
   - **Description**: Allows the user to reload the plugin configuration without restarting the server.
   - **Associated Command**: `/craftengineconverter reload` or `/cec reload`

3. `craftengineconverter.command.convert`
   - **Description**: Permits the user to initiate the conversion process for supported plugin configuration files into the CraftEngine format.
   - **Associated Command**: `/craftengineconverter convert [<plugin_name>] [<type>] [--dryrun] [--threads=<number>]`
